# 常用工具类
- 每一个Android开发者在日常开发中都会积累一些自己的代码片段。
- 本项目就是收集一些日常开发中的常用工具类,减少功能性代码的重复编写,方便快速开发。
- 通过封装一层工具类的好处是以后只需要修改工具类的实现，就可以替换整体的框架。
- 参考了众多项目并加以改进和总结,如有疑问,请来信说明。
